## Set up the environment

1. Create a virtual environment:
   *Using Python 3.13.2
   ```bash
   python -m venv venv
   ```

2. Activate the virtual environment:
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```bash
     source venv/bin/activate
     ```
3. Install the required packages:

   ```bash
   pip install -r requirements.txt
   ```
